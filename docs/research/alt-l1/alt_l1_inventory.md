# Alternative L1 Blockchain Inventory

## Major Alternative L1 Blockchains (by Market Cap, 2022-2025)

### Top Tier (by adoption/market cap)

1. **Solana** - High throughput PoS
   - Explorer: Solscan, Solana Beach, Explorer.solana.com
   - APIs: Public RPC nodes, Solscan API, SolanaFM

2. **Cardano (ADA)** - PoS with UTXO model
   - Explorer: Cardanoscan, Cardano Explorer, Blockfrost
   - APIs: Blockfrost API, Cardano GraphQL

3. **Polkadot (DOT)** - Multi-chain with parachains
   - Explorer: Subscan, Polkascan
   - APIs: Subscan API, Substrate RPC

4. **Avalanche (AVAX)** - Multi-chain PoS
   - Explorer: Snowtrace (Etherscan-like)
   - APIs: Avalanche API, Snowtrace API

5. **Algorand (ALGO)** - Pure PoS
   - Explorer: AlgoExplorer, Pera Explorer
   - APIs: AlgoExplorer API, Algorand Indexer

6. **Cosmos (ATOM)** - IBC hub
   - Explorer: Mintscan
   - APIs: Mintscan API, Cosmos REST API

7. **Near Protocol (NEAR)** - Sharded PoS
   - Explorer: NearBlocks
   - APIs: NEAR RPC, NearBlocks API

8. **Tezos (XTZ)** - PoS with on-chain governance
   - Explorer: TzStats, TzKT
   - APIs: TzKT API, TzStats API

9. **Aptos** - Move-based (since late 2022)
   - Explorer: Aptos Explorer, Aptoscan
   - APIs: Aptos REST API

10. **Sui** - Move-based (since 2023)
    - Explorer: Sui Explorer, SuiScan
    - APIs: Sui RPC

## Research Priority Order

1. Solana (highest tx throughput, vibrant DeFi)
2. Avalanche (EVM-compatible, DeFi focus)
3. Cardano (established, UTXO model)
4. Algorand (high performance)
5. Polkadot (unique multi-chain)
6. Cosmos (IBC ecosystem)
7. Near (sharding approach)
8. Tezos (established chain)
9. Aptos/Sui (newer chains, may lack 2022 data)
