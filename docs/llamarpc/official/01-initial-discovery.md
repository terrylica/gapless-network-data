# LlamaRPC Initial Discovery

**Date**: 2025-11-03
**Source**: https://llamarpc.com/

## What is LlamaRPC?

LlamaRPC (also known as LlamaNodes) is a blockchain RPC provider offering both free public endpoints and premium paid services. It operates as "a reliable public good built for the decentralized world."

## Maintainer & Trust Level

- **Organization**: LlamaCorp family
- **Transparency**: Open source project with transparent operations
- **Community**: Part of the DefiLlama ecosystem (implied by branding)
- **Trust Indicators**:
  - Public GitHub repository (llamanodes/web3-proxy)
  - Open source infrastructure
  - Medium blog presence
  - Listed on major aggregators (Alchemy Chain Connect, ChainList)

## Supported Chains

Primary chains confirmed:

- Ethereum (https://eth.llamarpc.com)
- Polygon (https://polygon.llamarpc.com)
- Base
- BNB Chain
- Goerli (testnet)

Note: "New chains are added regularly"

## Rate Limits

**Free Tier**: 50 Requests per Second (RPS)
**Premium Tiers**: Unlimited Requests with auto-scaling

## Key Features

1. **Privacy-first**: Web3 sign-in and crypto payments (no KYC)
2. **Archive data access**: Historical blockchain state retrieval available
3. **No contracts**: Pay-as-you-go model
4. **Flexible payments**: Stablecoins and credit cards accepted
